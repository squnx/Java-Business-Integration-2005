TIBCO® General Interface is a graphical user interface (GUI) framework and accompanying visual development environment that enables you to quickly create and easily deploy highly interactive, rich Web-based interfaces. Equipped with a library of extensible and high-performance GUI, data management, and communication components, TIBCO General Interface makes it possible to give browser-based interfaces the features and performance characteristics typically found only in thick-client GUIs. By using the asynchronous communications, Javascript, XML, and DHTML ("AJAX") capabilities present in the ubiquitous Web browser, TIBCO General Interface allows you to deliver rich internet applications (RIAs) with no applets, plug-ins, or installation of client/server frameworks.
