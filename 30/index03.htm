SUN JAVA BUSINESS INTEGRATION TECHNOLOGY CD-ROM
Solaris and Windows Version

RUNNING THE CD:
Double-click the file called "index.htm." If the file does not open, drag and drop into an open browser, either Mozilla, Firefox, Internet Explorer, or Netscape Navigator browser. The CD auto launch functionality is available on PC platform.

SYSTEM REQUIREMENTS:
- This CD works with Mozilla 1.7 and above, Firefox 1.0 and above, Netscape Navigator 7.0 and above, and Internet Explorer 6.0 and above.
- If you don't have a browser, please click on the Install folder and select a browser compatible with your system.
- For best results, use a machine with at least 512 MB RAM and a color monitor with 800 x 600 or higher resolution, capable of showing 16-bit (Hi Color setting).

INSTALLING JBI SDK FROM CD:
To Install the JBI SDK from CD, click on the Install buttons on the SDK landing page. Save the zipped file to your desktop.
- If you are on Solaris platform, locate and unzip the JBI75sol.zip file.
- If you are using Windows platform, locate and unzip the JBI75win.zip file. And then, double click the file called "install.bat" to start the installation.
